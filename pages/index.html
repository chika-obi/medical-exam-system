About MedEx

MedEx is a secure, auditable, and accessible online exam platform designed for medical education. It supports randomized questions, per-student shuffling, webcam proctoring, detailed exam logs, and instant result analytics — all with an easy-to-use admin interface.

Core Features

Secure Proctoring

Webcam snapshots, activity logging and event markers ensure integrity.

Randomized Delivery

Unique question and option shuffles per student reduce collusion.

Instant Grading

Automated scoring with per-question analysis and exportable reports.

Admin Controls

Schedule exams, set durations, review proctor logs, and export CSVs.

Responsive UI

Optimized for desktop and mobile devices.

Local-First Mode

Option to run with browser localStorage for small deployments or testing.

Frequently Asked Questions

Is MedEx secure?
+
Yes — webcam snapshots, browser activity logging and randomized delivery reduce fraud.
Does it work offline?
+
MedEx supports localStorage mode for offline testing, but live proctoring requires network.
Can I export results?
+
Yes — admins can export CSVs of exam results and proctor logs from the admin dashboard.
